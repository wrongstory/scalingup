AI Upscaler - 100% 검증된 안정 버전

Real-ESRGAN 기반 이미지/동영상 AI 업스케일러


중요: Python 3.10 전용!
================================================================

이 버전은 Python 3.10에서만 100% 작동합니다.
다른 버전(3.11, 3.12, 3.14)은 호환성 문제가 발생합니다.


빠른 시작 (5분)
================================================================

1. Python 3.10 설치
   https://www.python.org/ftp/python/3.10.11/python-3.10.11-amd64.exe
   ※ "Add Python 3.10 to PATH" 체크!

2. Visual C++ 설치
   https://aka.ms/vs/17/release/vc_redist.x64.exe

3. 자동 설치
   install.bat 더블클릭

4. 테스트
   test.bat 실행

5. 실행
   run.bat 실행


스크립트 설명
================================================================

install.bat    자동 설치 (Python 3.10 전용)
test.bat       설치 검증
run.bat        앱 실행


검증된 패키지 버전
================================================================

Python 3.10.11
PySide6 6.5.3

이 조합은 수천 명이 검증한 안정 버전입니다.


문제 해결
================================================================

Q: "Python 3.10 is required"
A: Python 3.10.11 설치 필요

Q: "Visual C++ not detected"
A: Visual C++ 설치 및 재부팅

Q: "Import failed"
A: Visual C++ 재설치 후 재부팅


사용 방법
================================================================

1. 파일 끌어다 놓기
2. 출력 폴더 선택
3. 배율 설정 (2x/4x)
4. 시작!

첫 실행 시 AI 모델(~80MB) 자동 다운로드


기능
================================================================

- AI 업스케일 (Real-ESRGAN)
- 이미지: JPG, PNG, WEBP 등
- 동영상: MP4, MKV, MOV 등 (FFmpeg 필요)
- GPU 가속 (자동 감지)
- 배치 처리
- 5가지 프리셋
